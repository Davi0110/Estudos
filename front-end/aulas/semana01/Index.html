Introdução ao HTML

O que é HTML

HTML é a linguagem padrão usada para criar páginas da web. Ele funciona como a estrutura de um site, organização de textos, imagens, links e outros elementos.

O que é e como funciona o VScode?

O Visual Studio Code é um editor de código leve desenvolvido pela Microsoft. Ele permite escrever, editar e organizar códigos em várias linguagens.
